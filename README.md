The goal of this project is to segment mall customers based on their demographics and spending behavior using unsupervised learning.
The analysis identifies key customer groups to help businesses target the right audience with personalized marketing strategies.

Key Steps

Loaded and cleaned the Mall_Customers.csv dataset.

Performed Exploratory Data Analysis (EDA) using Pandas, Seaborn, and Matplotlib.

Implemented K-Means clustering to group customers by spending habits and income.

Evaluated model performance using Elbow Method and Silhouette Score.

Built an interactive Streamlit dashboard for exploring clusters, visualizations, and insights.
